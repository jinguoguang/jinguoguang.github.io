Go语言学习系列之编译环境安装、配置

约 451 字大约 2 分钟

Go教程

2024-03-02

1.下载Go安装包和配置GOROOT、GOPATH

在 GoLang中国，可以下载该版本，如图所示。根据自己电脑的系统平台选择不同格式的Go安装包。下载指定版本可以在 Archived versions中查看。

2. 下载Go安装包

下载完成之后，点击安装一直下一步直到安装完成。

3. 配置GOROOT和GOPATH

GOROOT是安装包bin\go.exe所在的路径，GOPATH是Go工具包所在的路径(需要自己创建一个文件夹来存放Go项目)。

右击我的电脑图标打开高级系统设置，按图(1),图(2),图(3)操作：

添加GOROOT用户变量,如：D:\soft\Go,这里为Go的安装路径

我们可以看到用户变量已经默认创建了 GOPATH，只需点击修改就行。如：GOPATH = D:\soft\GoProject

如下图所示点击 Path，添加如下环境变量：

 %GOROOT%\bin
 %GOPATH%\bin

4. 配置Go代理和Go包校验方式

按住 WIn+R键，输入cmd命令，在打开的DOS 输入框中依次输入以下命令:

go env -w GOPROXY=https://goproxy.cn,direct
go env -w GOSUMDB=sum.golang.google.cn

1.4 验证Go环境

在 Cmd窗口输入go env命令查看具体信息，如输出下图所示信息，则证明配置成功。
