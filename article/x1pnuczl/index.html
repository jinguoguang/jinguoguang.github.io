唯一索引

唯一索引是指索引列的值不能重复，但允许有空值。

唯一索引的创建语法如下：

CREATE UNIQUE INDEX index_name ON table_name (column_name);

主键索引

主键索引是指索引列的值不能重复，并且不允许有空值。 主键索引的创建语法如下：

CREATE PRIMARY KEY index_name ON table_name (column_name);
