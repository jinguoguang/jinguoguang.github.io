Nginx安装部署

约 218 字小于 1 分钟

Nginx

2024-03-02

一、下载源码包

从Nginx官网下载所需源码包，稳定版本下载：https://nginx.org/en/download.html

本文以nginx-1.26.1版本作为示例

二、安装依赖库

使用以下命令查看是否安装所需依赖包，执行后有输出结果代表已经安装过了

yum list installed | grep "pcre-devel"

安装依赖库（如已经安装请忽略），执行以下命令

yum -y install gcc pcre-devel zlib-devel openssl openssl-devel

提示

如果yum下载比价慢的话，请自行参考【yum源配置，看这一篇就够了！】，配置yum国内镜像

./configure --prefix=/usr/local/soft/nginx --with-http_stub_status_module --with-http_ssl_module --with-pcre=/usr/local/soft/pcre-8.35
