函数定义

func 函数名(参数列表) (返回值类型) {
    // 函数体
}
  • func 是定义函数的关键字。
  • 函数名 是你给函数起的名字，遵循驼峰命名法。
  • 参数列表 是传递给函数的值，可以有多个参数，并指定每个参数的类型。
  • 返回值类型 是函数执行后返回的数据类型，如果函数没有返回值，则省略此部分。
  • 函数体 包含了函数的具体实现逻辑。

示例

1.不带参数和返回值的函数

func sayHello() {
    fmt.Println("Hello, world!")
}

2.带参数不带返回值的函数

func greet(name string) {
    fmt.Printf("Hello, %s!\n", name)
}
